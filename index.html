Información del Corrector
¿Cómo funciona?
  1. Pega tu código HTML (y CSS inline si tienes) en el editor
  2. Presiona "Analizar" para validar el código
  3. Revisa los resultados en el panel derecho
  4. Haz clic en los hallazgos para ver el código resaltado
Reglas de Validación
  • Único H1 por documento
  • LI dentro de UL
  • href solo en etiquetas A
  • src solo en IMG
  • Botones dentro de formularios
  • Prohibidas las tablas
  • Evitar etiquetas visuales
  • Preferir semántica HTML5
  • Estructura NAV > UL > LI > A
  • Estructura mínima requerida
  • Nombres de archivo válidos
  • Imágenes ≤ 500KB
  • Título del documento
  • Codificación UTF-8
  • Medios con controles
  • Documentos con nombres válidos
Puntaje

Aprobación: ≥ 60 puntos

Aciertos: +1 punto por regla

Errores: -2 puntos por regla
